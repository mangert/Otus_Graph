# Graph Representations

Учебный проект по реализации различных способов представления графов на C++.

## Состав проекта

### Заголовочные файлы (`headers/`)
- `IGraph.h` - базовый интерфейс для всех представлений
- `graph_factory.h` - функции для создания графов из списков вершин и рёбер

### Реализации представлений (`headers/`)
1. `GraphEnumeration.h` - перечисление множеств
2. `GraphAdjMatrix.h` - матрица смежности
3. `GraphIncMatrix.h` - матрица инцидентности
4. `GraphEdgeList.h` - перечень рёбер
5. `GraphAdjVectors.h` - векторы смежности
6. `GraphAdjArray.h` - массивы смежности
7. `GraphAdjList.h` - списки смежности
8. `GraphStructured.h` - структура с оглавлением
9. `GraphVertexEdgeList.h` - список вершин и список рёбер

### Исходные файлы (`src/`)
- `main.cpp` - точка входа, запуск тестов
- `bipartiteGraph.cpp` - тестирование двудольного графа А(3,4)
- `test.cpp` - комплексные тесты всех представлений

### Отчёт (`report/`)
- `output.txt` - результаты тестирования
- `graph.pdf` - изображение двудольного графа
- `report.md` - подробный отчёт о выполненной работе

## Особенности

- Шаблонная архитектура с поддержкой произвольных типов вершин и информации о рёбрах
- Параметр `Directed` для выбора типа графа (ориентированный/неориентированный)
- Индексные структуры для быстрого поиска вершин и рёбер
- Полное покрытие тестами всех 9 представлений
